### About me

I write TypeScript by day, argue with LLMs by night, and somehow both end up in production. 🚀

Full-stack engineer who has spent years making AWS serverless infrastructure do things it
probably was not designed for. Event-driven architecture with Lambda, EventBridge, and SQS.
Infrastructure as code with SST and AWS CDK. Observability with Datadog because "it works on
my machine" is not a monitoring strategy. 📊

Currently deep in AI engineering territory. Building RAG pipelines, real-time streaming UIs,
and agentic workflows. Also spent time telling Claude Code exactly what it is and is not allowed
to do — turns out AI needs boundaries too. 🤖

Now exploring LangGraph and MCP servers because apparently one personal AI project was not enough. ⚡

I care about clean systems, code that does not wake you up at 3am, and shipping things that
actually hold up when real users show up.
